# Changelog

Все заметные изменения проекта документируются в этом файле.
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
проект придерживается семантического версионирования.

## [1.0.0] — 2026-06-24

### Добавлено
- Учёт инцидентов с отметками времени по этапам жизненного цикла
  (компрометация → обнаружен → взят в работу → эскалирован → изолирован →
  триаж → отчёт → заключён).
- Авторасчёт метрик времени с медианой, средним и p90: MTTD, MTTA, время до
  эскалации, время до изоляции хоста, триаж, срок отчёта, заключение L3, MTTR.
- Доли и исход: поймано своими детектами, TPR, FPR, доля возвращённых на
  доработку, найден root cause, SLA взятия в работу, SLA изоляции.
- Автоматический расчёт соблюдения SLA из порогов по severity.
- Редактируемый каталог типов инцидентов с привязкой к MITRE ATT&CK
  (добавление / изменение / удаление прямо в интерфейсе).
- Разбивки по типам, severity и технике MITRE; группировка по неделям,
  месяцам и кварталам.
- Экспорт JSON / CSV, импорт JSON. Локальное хранение в IndexedDB, без сети.
- Единый файл без зависимостей, работает офлайн.
